# メンデル遺伝・減数分裂シミュレーター AI 指示書

## 0. 共通事項

本仕様書は、既存の参考プロジェクト群と同じ設計思想・運用前提で動作する Web 教材の作成指示書である。個別機能に入る前に、全テーマ共通の前提を以下にまとめる。以降の章は、この共通事項を前提として本テーマ固有の要件のみを記述する。

### 0.1 参考プロジェクト

- [GEO-LEARNER](https://takayuki-minagawa.github.io/GEO-LEARNER/) — 観測視点切替 + 学習モジュールの三層構成
- [weather-front-sim](https://takayuki-minagawa.github.io/weather-front-sim/) — 時系列操作 + 観測値グラフ + 学習パネル + クイズ
- [web-planetarium](https://takayuki-minagawa.github.io/web-planetarium/) — 地点・時刻に応じた星空描画と i18n / テーマ切替

これらと同じ設計思想（操作パネル＋可視化ビュー＋学習パネル＋クイズ、i18n、ライト／ダーク切替、静的配信）を踏襲する。

### 0.2 共通仕様

- 静的フロントエンド構成とする（React + TypeScript + Vite を基本、軽量なら Vanilla JS + HTML + CSS も可）
- GitHub Actions により GitHub Pages へデプロイする
- 日本語 UI を基本とし、英語切替（i18n）に対応する
- ライト／ダークテーマ切替に対応する
- PC・タブレット・スマホ対応のレスポンシブ UI にする
- 画面は「操作パネル」「可視化ビュー」「学習パネル（解説＋クイズ）」の三層構成を基本とする
- 教育用途のコンテンツであり、実測・予報・診断など厳密な数値提供を目的にしない
- 公開リポジトリ前提の表現で README と仕様書を書き、個人情報・認証情報・秘密鍵は含めない

### 0.3 共通で再利用したいコンポーネント

- タイムスライダー、再生・一時停止・速度変更コントロール
- 学習パネル・クイズフレーム（状態に応じた解説文生成と正誤判定）
- i18n ロード機構、ライト／ダークテーマ切替
- 3D シーン基盤（Three.js + React Three Fiber）: 3D 系テーマで共用
- 2D グラフ・波形描画基盤（Canvas / D3.js）: 時系列グラフ系テーマで共用
- SVG 編集系 UI（図形選択・ドラッグ・履歴）: 図形操作系テーマで共用

### 0.4 学習パネルの共通方針

- 現在の状態・入力に対する解説を自動生成する
- 観察の着目点や誘導テキストを出す
- クイズモードを併設し、主要な誤解ポイントを問題化する
- 言語切替に追従して文言が再生成される構造にする

## 1. 目的

親個体の遺伝子型を設定すると、減数分裂のアニメーションを経て配偶子が作られ、受精によって子の遺伝子型・表現型の分布が確率的に現れる様子を観察できる Web 教材を作成する。

高校生物・生物基礎の「遺伝」「減数分裂」の単元で扱う内容を、Punnett 四角と動的アニメーションの両面から理解できる体験型教材とする。

- 優性の法則、分離の法則、独立の法則
- 対立遺伝子の挙動
- 減数分裂における染色体の分配（第一分裂・第二分裂）
- 独立遺伝と連鎖・組換え（発展）
- F1, F2 世代の分離比

## 2. 開発方針

### 2.1 基本方針

- 静的フロントエンド Web アプリ
- GitHub Pages 公開
- PC／スマホ対応、日英切替、ライト／ダーク

### 2.2 推奨技術スタック

- React + TypeScript + Vite
- 描画: SVG（染色体アニメーション）＋ Canvas（多数世代シミュレーション用）
- スタイリング: Tailwind CSS または CSS Modules

### 2.3 画面構成

- **減数分裂ビュー**: 細胞内の染色体が複製・対合・分離する流れをアニメーション
- **交配ビュー**: 親 P 世代の遺伝子型、配偶子、Punnett 四角、子の表現型分布
- **グラフパネル**: 大数試行時の分離比プロット
- **学習パネル**: 解説・クイズ

## 3. 必須機能

### 3.1 遺伝子設定

- 対立遺伝子の名前（A/a, B/b, R/r など）を自由に追加
- 優性・劣性の関係を明示
- 表現型マップ（AA=赤, Aa=赤, aa=白 など）を編集
- プリセット
  - エンドウの種子色（黄 Y / 緑 y）
  - エンドウの形（丸 R / しわ r）
  - 血液型（ABO 式）
  - 2 遺伝子独立遺伝（YR, Yr, yR, yr）

### 3.2 親個体の設定

- 父・母の遺伝子型をドロップダウンで設定（AA × aa, Aa × Aa 等）
- 各個体の見た目を色や形でプレビュー

### 3.3 減数分裂アニメーション

- 細胞内で染色体が複製される様子
- 第一分裂（相同染色体の分離）と第二分裂（姉妹染色分体の分離）を段階的に描画
- アニメーション速度を調整可能
- 染色体ごとに色分けし、どの組合せが配偶子に入るか追いやすくする
- 独立遺伝の場合、異なる組合せが等確率で生じることを可視化

### 3.4 Punnett 四角

- 父母の配偶子を行列状に配置
- 交配結果を格子内に自動表示
- 遺伝子型と表現型の比を計算し、円グラフまたは棒グラフで表示

### 3.5 確率的試行モード

- F1 × F1 交配で、数十〜数万子個体を確率的に発生させる
- 試行回数を変えると理論比に収束する様子を表示
- 子世代の表現型分布をヒストグラム化

### 3.6 学習パネル

- 現在の交配結果に対する解説
- 優性／劣性、分離比、独立遺伝の意味
- クイズ
  - 配偶子の種類を答える
  - 分離比を予想
  - ABO 式血液型の子の可能性判定

## 4. あると良い機能

- 連鎖と組換え（発展）: 2 つの遺伝子が同じ染色体にある場合
- 伴性遺伝（発展）
- 家系図入力モードで親族 3 世代の遺伝子型推定
- 多数世代シミュレーション（ハーディ・ワインベルグの体験）
- URL クエリで条件共有
- PWA 対応

## 5. データ・計算設計

### 5.1 遺伝子型モデル

- `Allele`: 名前、優性レベル
- `Gene`: 対立遺伝子リスト、表現型マップ
- `Genotype`: Gene ごとの allele ペア
- `Gamete`: 配偶子としての allele 組合せ

### 5.2 減数分裂ロジック

- 独立遺伝モード: 各 Gene ごとに独立に allele をランダム選択
- 連鎖モード: 同じ染色体にある Gene は組換え率に従って交換

### 5.3 Punnett 自動生成

- 配偶子の全パターン列挙
- 交配結果の遺伝子型と表現型を集計

### 5.4 プリセット

`geneticsPresets.ts` に古典的な交配例を定義。

---

## 6. アニメーションとビジュアル

- 染色体は棒状の 2 本線で描画し、動的に対合・分離・移動
- 細胞境界は楕円で表示し、細胞質分裂で分かれる
- 配偶子は小さな細胞アイコンとして右側にストック
- Punnett 四角はセルホバーで該当する遺伝子型をハイライト
- 試行モードでは子個体を次々にベルトコンベアのように流す
